## üìå Project Overview

**CodeNest** is a comprehensive web platform designed to streamline coding education and online assessments.

### Panels
- **Super Admin** ‚Äì Platform governance & monetization  
- **Admin (Institutions / Teachers)** ‚Äì Batch, contest & student management

This structure ensures enterprise-level control, institutional scalability, and a competitive learning environment.
